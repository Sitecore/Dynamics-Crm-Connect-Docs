Home
===================================================
Welcome to the Sitecore Connect for Microsoft Dynamics 365 for Sales
Developer Guide. This documentation covers how to 
customize and extend the tool available for 
synchronizing data between Microsoft Dynamics 365 for Sales and 
Sitecore Experience Platform 9.0.1.

.. note::

    This guide assumes that you have installed Sitecore 
    Connect for Microsoft Dynamics 365 for Sales on your Sitecore server 
    and that you have created a tenant. The installation
    guide covers both of these requirements. It is 
    available on the Sitecore Developer Portal.
