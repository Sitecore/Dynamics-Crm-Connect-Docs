Adding Custom Data
===================================================
Sitecore Connect for Microsoft Dynamics 365 for Sales comes pre-configured 
to synchronize contact data between Microsoft Dynamics 365 for Sales and 
Sitecore Experience Platform. 

The product allows you to add other data from either 
Microsoft Dynamics 365 for Sales or Sitecore Experience Platform. This 
includes both data that is available by default and
custom data you have added in your environment.

.. note::

    Finished code and installation packages are available in 
    `GitHub <https://github.com/Sitecore/Sitecore.DataExchange.Examples/>`_.
